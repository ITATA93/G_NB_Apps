# Archive - Scripts Obsoletos UGCO

Esta carpeta contiene scripts obsoletos de UGCO que fueron reemplazados por versiones más nuevas pero se mantienen temporalmente por precaución.

## Política de Archivado

- Los scripts aquí **NO deben usarse** en producción
- Se mantienen por 1-2 sprints antes de eliminarlos definitivamente
- Si un script archivado no se ha necesitado después de 2 meses, puede eliminarse permanentemente
