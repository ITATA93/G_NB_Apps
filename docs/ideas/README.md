# Ideas de Aplicaciones

Repositorio de ideas y propuestas de aplicaciones para implementar en NocoBase.

## Estructura de Propuestas

Cada idea debe documentarse en un archivo separado con el formato:

```
NOMBRE-APP.md
```

### Template para nuevas ideas

```markdown
# [Nombre de la Aplicación]

## Resumen
Breve descripción del propósito de la aplicación.

## Problema que resuelve
- Punto 1
- Punto 2

## Usuarios objetivo
- Rol/Perfil 1
- Rol/Perfil 2

## Funcionalidades principales
1. Funcionalidad A
2. Funcionalidad B
3. Funcionalidad C

## Colecciones/Tablas necesarias
| Tabla | Descripción |
|-------|-------------|
| tabla_1 | ... |
| tabla_2 | ... |

## Integraciones requeridas
- [ ] ALMA (HIS)
- [ ] SIGO
- [ ] Otro sistema

## Prioridad
- [ ] Alta
- [ ] Media
- [ ] Baja

## Estado
- [ ] Idea inicial
- [ ] En evaluación
- [ ] Aprobada
- [ ] En desarrollo
- [ ] Completada

## Notas adicionales
...
```
