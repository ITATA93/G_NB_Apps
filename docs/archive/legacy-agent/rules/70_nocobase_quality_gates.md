# 70 - Quality gates (DoD tecnico)

Antes de declarar "terminado":
1) API Key funciona (prueba con endpoint permitido)
2) Roles/permisos minimos creados y probados
3) Collections y fields del blueprint existen
4) UI: menu/pagina principal accesible para roles esperados
5) Workflows (si aplica): habilitados y probados
6) Evidencia: artifact con checklist PASS/FAIL y notas
