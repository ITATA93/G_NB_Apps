# Documentación de Colecciones

Esta carpeta contiene la documentación detallada de cada colección (tabla) de la aplicación.

## Estructura de Documentación

Cada colección debe tener un archivo markdown con el siguiente formato:

```
[NOMBRE_COLECCION].md
```

## Template de Documentación

Ver [TEMPLATE_COLECCION.md](TEMPLATE_COLECCION.md) para el formato estándar.

- Nombres de archivo en minúsculas con guiones bajos
- Un archivo por colección
- Incluir diagrama de relaciones si aplica
- Documentar todos los campos con descripción clara
- Incluir ejemplos de valores
